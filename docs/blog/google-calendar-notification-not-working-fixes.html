Google Calendar Notification Not Working? 9 Fixes That Actually Work

Direct answer: When Google Calendar notifications stop working, it’s usually caused by browser notification settings, OS Focus/Do Not Disturb, permission resets, or multiple devices/accounts. Start with the 3 quick checks below, then run the full checklist.

3 quick checks (most common)

1) Chrome notifications allowed

Direct answer: If Chrome can’t send notifications, Calendar can’t notify you.

  • Chrome Settings → Site settings → Notifications
  • Make sure notifications are allowed (and not blocked)

2) OS Focus / Do Not Disturb

Direct answer: Focus modes commonly silence Calendar without telling you.

Check Focus / Do Not Disturb settings and allow Chrome.

3) Signed into the correct Google account

Direct answer: Wrong account = wrong settings = “no notifications.”

Full fix checklist

  1. Confirm the event has a notification set
  2. Re-enable notifications permission for calendar.google.com
  3. Temporarily disable ad blockers / privacy extensions and test
  4. Check if notifications are going to another device
  5. Clear site data for calendar.google.com (last resort)
  6. Confirm time zone settings (late/early can feel “broken”)
  7. Ensure background restrictions aren’t silencing alerts
  8. Look for grouped/collapsed notifications
  9. Add a redundancy plan for critical meetings
